Pack Cloud Storage

A low cost platform for users with personal data

PROJECT OVERVIEW:

The available cloud storage platforms in the market are mainly focusing on the businesses. Even with the individual available plans, the users are complaining about the plans cost for their personal data storage. Pack provides a comfortable and user-friendly platform for everyone with personal data at a lower cost. There was a need for a personal easy-to-use platform in the market which I was fortunate to help with the design and development. Pack has a special respect toward the students and offer them a special incentive. This does not mean non-students do not receive any benefits. There is a referral programs for anyone to get free extra storage space.

I.Why are loud services becoming more popular?

Cloud services are becoming popular because they can reduce the cost and complexity of owning and operating computers and physical storages. Growing demand for universal data accessibility, sync technology and security, increases in cloud adoption across industry and society.

II.What percentage of companies use the cloud?

93% of organizations use cloud-based IT services according to Spicework Press, which is a siginificant percentage.

III.What percentage of online adult use the cloud?

A statistic on Statista presents information on cloud storage usage among online consumers in the United States, sorted by age group. During the May 2017 survey it was found that 61 percent of online consumers aged 18 to 29 years were using online cloud storage services.


According to the collected data, the companies are the dominant cloud storage users compared to indivisual users.

Competitive Analysis


SWOT
Aside from the user survey, competitive analysis helped me to determine the strengths and weaknesses of the competitors within the market. It provided me to spot opportunities for differentiating my cloud storage. I’ve selected Google Drive, Dropbox and Evernote for analysis.

To observe the comprehensive findings on SWOT analysis click HERE.


Based on the analysis above and survey data, I needed to answer the question of “How a new competitor enter the marketplace?”.



I. Cost vs storage space/data storage space for free version:
I believe the new competitor needs to provide the same level of services with lower/comparable price. It also needs to provide the users with higher data storage space for free version or add some interesting features to the free version of accounts. (It could be the features on the paid plans of available competitors)



II. Operating systems:
My recommendation for the new competitor is to cover the operating system gap existed in the market. So, it could offer a new service to the minority operating system users like windows phone or Linux/Unix users.



III. Data Organization:
If I could set an automatic and smart data organization feature, it could be an apportunity to enter the product into the market. For example, a user would like to have the videos separated from the images and documents. By adjusting the setting to the file type, the videos go to the specific folder for the videos or the uploaded image goes to the image folder directly right after being uploaded. This could save a few clicks for users.



IV. Integration with other apps in market/ supporting a broad file types:
To compete with the available products in the market, the new competitor should better have a broad integration with available applications out there. What might be interesting to the users is to customize the cloud platform app integration based on the users’ career or need.



V. Data migration among the cloud platforms:
If the new competitor provides an easy data migration tool for the users, it’ll help to absorb more users to bring their data into the new cloud storage.



VI. Referral program incentive:
My idea for the new competitor is to consider a referral program incentive for the users. They could also offer users free data storage space for migrating their data from any other cloud platform to their platform.



VII. A Combination of useful features in existing cloud storage platforms to the users:
In order to enter into the market, the new competitor could combine the interesting features of existing platforms. It also could offer users what they really need based on their budget. For example, if it’s important for the user to have a fast sync feature on the platform, the platform could be customized with that feature. The same could happen for the strong search feature.



VIII. The missing features on the mobile app:
If the new competitor would be able to blame that a user could access to the same level of services using any devices, that’s going to be a unique feature. So, it would be even possible for the user to do all the required tasks just using their cell phone or tablet.



IX. Collaboration features:
The new competitor collaboration features could be a comprehensive combination of available cloud storage platforms. It also could expand this feature to social media, URL with protection password, to comment on all types file, like/dislike the changes, to send notification if a file is opened/modified, to send a request to do task on the shared file, to receive the task on the shared file is all done, to lock the shared file/folder in case of emergency, and many more.
